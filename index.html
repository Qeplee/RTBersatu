🌺 Tentang SusiBungas

Satu Platform untuk Semua Warga

SusiBungas hadir sebagai solusi digital terintegrasi yang memudahkan Ketua RT dalam mengelola data warga, kegiatan, dan keuangan RT secara transparan dan real-time.

📱
Aplikasi Android untuk Ketua RT
Input data warga, laporan kegiatan, dan keuangan RT langsung dari smartphone
🌐
Website Publik untuk Semua Warga
Akses statistik desa, pengumuman, dan info RT kapan saja tanpa login
🔒
Dashboard Privat per RT
Setiap RT memiliki dashboard khusus yang hanya bisa diakses Ketua RT yang bersangkutan
